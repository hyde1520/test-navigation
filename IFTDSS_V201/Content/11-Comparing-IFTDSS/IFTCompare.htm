The IFTDSSInteragency Fuels Treatment Decision Support System. A web-based software integration framework that organizes and extends previously existing fire and fuelsCombustable materials that a fire consumes in order to burn. The modules in IFTDSS are organized to fulfill specific business needs, which have been defined as IFTDSS workflows. In some cases, only specific functional pieces of larger applications have been implemented. Many modules and software applications have been and will be added to IFTDSS.

Quality assurance tasks are performed to ensure the modules implemented in IFTDSS provide the same numerical values as the standalone desktop versions of the modules. Additionally, the functionality of the modules implemented in IFTDSS has been compared with the original standalone desktop applications. These comparisons were made to inform users about the functionality that is and is not included in the IFTDSS implementation.

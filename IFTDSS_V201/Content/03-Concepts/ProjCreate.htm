To create a project,

  1. Choose Create New Project.

  1. This opens the Create New Project page, where you enter the details for your project. Enter a descriptive project name, which is required. Everything else is optional, though the more information you include now, the better.

  1. Choose Next. Your project is saved and you are taken to a page where you choose the type of run you want to create.

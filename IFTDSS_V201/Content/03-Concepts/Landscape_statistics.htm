Landscape statistics feature

Your landscape features can be viewed in tabular format, or exported to Excel as a CSVComma-separated values. This is a way of formatting data in a file so that it can be read by different programs. Usually this type of file is produced by saving a database or spreadsheet in this format. In this format, the values for each row or record are separated by commas. spreadsheet. These features can aid in quantifiably comparing results from different treatments or runs. To view the landscape features in tabular format, select the Landscape Statistics button.

To learn more about this feature, watch the video or read the content below:

Next, select the statistics option on the top right of the dropdown menu.

Once the statistics button has been selected, you can select your parameter, and a specific polygon, or the entire dataset, then select Go.

   Note   

If you have made a change to the map and want to ensure it is reflected in the statistics, check the Force Recalculation box and select Go.

When your calculation has completed you will see a listing of the acres and percent landcover for your selected feature.

Note the Download CSV button that allows you to export the data to a csv file for further analysis.

Scrolling further down, the results are also displayed in graphical format

Spotting Distance (IFT-spot)

You can estimate the maximum distance from different possible sources where one can expect potential spot fires, then use that information when preparing a prescribed burn plan. Spotting Distance from a Burning Pile

Spotting distance from a burning pile is the maximum distance at which one can expect potential spot fires resulting from firebrands from a burning pile.

Spotting Distance from Torching Trees

Spotting distance from torching trees is the maximum distance at which one can expect potential spot fires resulting from firebrands from torching trees.

Input variables include terrain, wind speed, flame height, and tree characteristics. The model provides spotting distance from torching trees.

(Albini and Baughman, 1979; Chase, 1981)

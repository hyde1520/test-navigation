Ignitions and Barriers
Ignitions Ignitions can be points, lines, or polygons.  Ignitions can be drawn interactively using the draw polyline tool on the map toolbar in the Ignitions step.
Barriers Barriers can be points, lines, or polygons depending on the pattern of mouse button clicks used and either filled (unburnable) or unfilled (burnable). Only closed polygons can be filled with an unburnable interior. These can be drawn interactively using the draw polyline tool on the map toolbar in the Barriers step.

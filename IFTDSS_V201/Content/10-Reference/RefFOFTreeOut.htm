Name Units Brief Description
Average Mortality Percentage Average probability (percentage) of tree mortality.
Trees Killed An integer Total number of trees killed.
Average DBHTree diameter at breast height. Because trees are generally wider at the base and narrower higher up, the diameter at breast height is used to define the size of the tree. The definition of breast height can vary but is usually 4.5 feet above the ground. of Killed Trees Inches, centimeters Average tree diameter (DBH) of fire-killed trees.

Total Trees per Fire An integer Total number of pre-fire live trees.

Basal Area: post-Fire Square feet per acre / square meters per hectare The horizontal land area covered by live tree boles after burning.

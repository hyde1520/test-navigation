Using the options in the Select FCCS Fuelbeds window, you can create one or more data sets.

To create a fuelbed data set,

  1. Access the Data Tab, then access the Fuelbed Data tab.
  2. Choose Create a Data Set of FCCS Fuelbeds. This opens the Select FCCS Fuelbeds window, which has a number of options. In this window, you can select a fuelbed data set from the list of all available FCCS fuelbedsA fuelbed is an area of similar fuel characteristics within a unit. A fuelbed contains one or more fuelbed strata (e.g., canopy, shrubs, etc)., or you can narrow the data sets first, then select from the remaining options.
  3. If you wish to narrow your data set, under Select FCCS Fuelbeds, select an ecoregion from the Ecoregion drop-down list. If you choose anything other than Any Ecoregion, the list of available fuelbeds in the FCCS Fuelbeds section changes so that it only includes fuelbeds from the selected region.
  4. You can then consecutively select a vegetation form, a cover type, a forest structural class, and a change agent. Each subsequent selection narrows the selection list further. Note that if you are trying to narrow the list to single fuelbed, you might not need to select from every selection box.
  5. Under FCCS Fuelbeds, select one or more fuelbeds.
  6. After making your selections, you can create one or more fuelbed data sets. You can create a single fuelbed data set from one or more selected fuelbeds, or you can create single fuelbed data sets with one or more fuelbeds.
    • If you would like to create a single fuelbed data set with one or more fuelbeds, name your data set, then choose Create Data Set.
    • If you would like to create individual fuelbed data sets from one or more selected fuelbeds, you have the option of naming your data sets (name of data set_fuelbed number). For example, if you assigned the name conifer to the data set, and you had selected fueldbeds 181 and 208, then IFTDSS creates two data sets named as follows: conifer_181 and conifer_208. Or you can leave the data set name blank, and IFTDSS will assign a fuelbed file name to each data set (FB_0181_FCCS.xml, FB_0208_FCCS.xml). After either naming or not, choose Create Separate Data Sets.
  7. Your data set will be created, and you are returned to the Fuelbed Data tab.

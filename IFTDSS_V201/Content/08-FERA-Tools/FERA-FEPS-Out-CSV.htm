Exported Hourly Consumption and Emissions Parameters

When you export the hourly consumption and emissions data, they are saved in a file that uses the CSVComma-separated values. This is a way of formatting data in a file so that it can be read by different programs. Usually this type of file is produced by saving a database or spreadsheet in this format. In this format, the values for each row or record are separated by commas. format. The parameters in that file are described in the following table.

   Note   

The number of parameters listed in the CSV-formatted file will be much larger than shown in this table. Also, in this table, the word Pollutant represents specific pollutant names, such as CO or CH4. So, for example, you won't see "Pollutant - Flaming" in the output file, but you might see "CO - Flaming."

Pasquil Stability NA The Pasquil stability class of the atmosphere, calculated based on midflame (surface) wind speed and solar radiation (for daytime), or surface wind speed and percent cloud cover (for nighttime). There are six Pasquil Stability categories: A through F. The Unstable classes (A, B, and C) occur during the daytime. The Stable classes (E and F) only occur at night. The Neutral class D can occur during the day or night.  Class A is ‘Very Unstable’ and corresponds to hot, calm days, which leads to the greatest amount of dispersion. A plume of smoke is broken up and spread widely with ‘A’ Stability. Class D is ‘Neutral’.  It corresponds to windy days or the transition times of dawn and dusk.  This is the most frequently occurring stability class. Class F is ‘Very Stable’ and corresponds to nights with low winds.

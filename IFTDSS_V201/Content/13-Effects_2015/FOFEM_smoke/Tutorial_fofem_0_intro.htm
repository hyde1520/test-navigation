   Note   

We will begin by setting up a projectA project consists of one or more units that are organized and managed as a single endeavor. area for this example, which contains several fuel cover types. To run IFT-FOFEM with more than one fuel type, you must run several simulations within the run, using one fuel type per scenario. This can be done as batch, so that one run produces results for numerous simulations. The results for IFT-FOFEM are displayed in tabular format, and summarized by charts, but not geospatially.

For this tutorial we will run IFT-FOFEM to compare the emissions that may result from consumption of fuel in an untreated stand, with those of a stand treated with prescribed fire.

Editing Data Using Data Studio

To edit data using data studio, navigate to the Data tab.

To open data for editing, click on the cog icon to the right of the dataset, and select edit

   Note   

Because Data Studio opens in a new browser window, youâ€™ll need to disable pop-up blockers.

Once the edit option is selected, the data will appear in a new screen

From this screen you may use the mapping toolbar to:

  • Point Edit: edit one pixel at a time
  • Advanced Edit: edit pixels across the entire run area
  • Polygon Advanced Edit: edit pixels within a user-drawn polygon

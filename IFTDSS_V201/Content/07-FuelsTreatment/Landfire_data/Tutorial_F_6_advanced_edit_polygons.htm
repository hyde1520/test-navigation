Polygon Advanced Edit: edit pixels within a user-drawn polygon

Using the Polygon Advanced Editing feature allows you to edit multiple pixels at a time within a polygon. After beginning a session in data studio, use the following steps to point edit:

  1. Select the Polygon Advanced Editing tool
  2. A dialogue box will appear on the left, ensure you have the polygon/feature that you want to alter specified
  3. Specify the conditions under which pixels should be changed, and what changes should be made.

The criteria under which a pixel may be changed allow for flexibility. You may specify to select pixels that are:

  • equal to
  • less than
  • greater than
  • equal to or less than

  • has no assigned value

It is also possible to modify pixels based on multiple criteria by selecting the add more criteria option

The same is true for the modification made to the pixels; to add modifications select the modify more values option

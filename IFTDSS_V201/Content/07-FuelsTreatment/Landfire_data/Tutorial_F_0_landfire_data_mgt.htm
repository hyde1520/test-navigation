Tutorial F: LANDFIRE Data Management in IFTDSS

This landscape-based tutorial covers the following:

  • How to upload or acquire LANDFIRE data.
    • Upload a landscape (.lcp) file and corresponding projection (.prj) file from your local machine.
    • Acquire landscape data from LANDFIRE for a specified area of interest using map tools in IFTDSS.
  • Editing LANDFIRE data in Data Studio.
    • Point edit – editing one pixel at a time.
    • Advanced edit – editing multiple pixels at once.
    • Polygon advanced edit – editing pixels within a user-drawn polygon.

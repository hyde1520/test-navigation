Point Editing

Point editing allows you to edit one pixel at a time. After beginning a session in data studio, use the following steps to point edit:

  1. Select the Point Edit tool
  2. Click on the pixel you would like to edit, and the Point Edit panel appears
  3. Edit the pixel data
  4. Click Save

Post Treatment Outputs

The Post Treatment Outputs window will allow the user to view both the pre and post treatment MTT runs as well as Differences (change) outputs.

You are able to see the treatment made a significant difference to fire spread within the treatment area. Perhaps one of the objectives of this treatment was to protect infrastructure along the highway to the north.

The Differences outputs in the left hand layers menu allow the user to quantify the amount of change the treatment has made on the landscape. Differences are expressed as a range of negative values (decrease in fire behavior) and positive values (increase in fire behavior). For the image below, we can see there are several places within our treatment polygon that show a negative Fireline Intensity Percentage (indicated in blue). Fireline intensity decreased in these areas from the Pre-Treatment to Post-Treatment fire behavior outputs due to the Squaw RX Burn. Fireline intensity decreased at varying rates across the treatment area and actually increased in some spots. Use the point edit to evaluate why Fireline intensity increased in some pixels post prescribed burn. Take a few moments to click on some pixles using the point edit tool, to bring up the attributes for individual pixels, noting by what percent some of then increase or decrease.

The Landscape Statistics tool can also be used to further analyze the pre/post-treatment differences. The Download CSVComma-separated values. This is a way of formatting data in a file so that it can be read by different programs. Usually this type of file is produced by saving a database or spreadsheet in this format. In this format, the values for each row or record are separated by commas. can be used to export the data into an Excel spreadsheet. The Chart context menucan be used to print the chart or download several varieties of images of the chart that can be used in other documents and reports.

Users may also examine values at an individual Pixel level using the Point Info tool . After selecting the point info tool click on the point of interest and the Point Info panel appears with a list of values for that point (individual pixel) including the output values.

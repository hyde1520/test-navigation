Exporting to Google Earth

To export data to Google Earth follow these steps:

  1. Access the Exports tab (located in the upper left panel).
  2. Select Download Output Data (KMLKeyhole Markup Language (KML), which is based on XML, is a file format used to display geographic data in an Earth browser (such as Google Earth, Google Maps, and Google Maps for mobile). For more information, see http://code.google.com/apis/kml/documentation/kml_tut.html.).
  3. Select Open with Google Earth in the pop-up window. (The pop-up window may look different depending on which browser you are using. Choose Open no matter what the pop-up window looks like.)
  4. Repeat these steps to Download Input Data (KML).

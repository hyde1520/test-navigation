On the upper toolbar, click the Point Edit tooland select the desired grid cell.

The Point Edit panel appears. From here, you can edit all input values. In this example, FM3: Tall Grass is selected from the Fuel Model dropdown list.

Some grid cells are labeled as FM8: Compact Timber Litter and FM4: Chaparral, which should be labeled as FM2: Timber Grass and Understory or FM9: Hardwood Litter. There is also a handful of cells with no data. These cells are edited one at a time.

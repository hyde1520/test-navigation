Appendices

The following Appendices are created once the Burn Plan is generated and exported into a Microsoft Word document:

  1. Maps
  2. Technical review checklist
  3. Complexity analysis
  4. Agency-specific job hazard analysis
  5. Fire behavior modeling documentation and/or empirical documentation
  6. Smoke Management Plan and Smoke Modeling Documentation (optional)

(USDA and U.S. DOI, 2014, p. 38)

Prescribed Burn Planning

All prescribed fires require an approved plan, called a prescribed burn plan, that must be followed when a burn is conducted. The prescribed fire burn plan is the legal document that provides an agency administrator with the information needed to approve a prescribed fire.

Top-Level Tasks

To access and run the fire behavior and fire effects modules within IFTDSS, or to run one or more modules,

  1. Determine your analysis goals and the type(s) of fire behavior and/or fire effects output information you wish to analyze.
  2. Create a new project.
  3. Create a new run.
  4. Select the Prescribed Burn Planning workflowA set of modeling pathways that provide access to fuels treatment analysis and planning tools in a stepwise and intuitive manner. Each workflow addresses a particular business need..
  5. Select the module(s) you wish to run. Depending on your needs, there are a number of different modules in IFTDSS that can be used to create a prescribed burn plan. The matrix on the Overview and Objectives page shows which modules are useful for which elements in the prescribed burn plan. The table on the Modules for Prescribed Burn Planning page provides an overview of the fire modules that are useful for prescribed burn planning.
  6. Configure the module and input the needed data. The inputs and outputs for the modules in the prescribed burn planning work flow are listed in the Reference section.
  7. View and interpret your results.
  8. Export your results from IFTDSS to your local computer.
  9. Edit your burn plan document, then print it as you would any Word document.

Other topics in this help provide more details about each of these steps. For step-by-step walkthroughs of the prescribed burn plan process, see the tutorials.

Preparing a Prescribed Burn Plan

This online tutorial illustrates how to use IFTDSSInteragency Fuels Treatment Decision Support System. A web-based software integration framework that organizes and extends previously existing fire and fuelsCombustable materials that a fire consumes in order to burn. The models in IFTDSS take fuelCombustable materials that a fire consumes in order to burn. The models in IFTDSS take fuel inputs such as trees, bushes, and grasslands. inputs such as trees, bushes, and grasslands. software applications to make fuels treatment planning and analysis more efficient and effective. to prepare a burn plan. Tools for preparing and generating a burn plan document in IFTDSS are based on the 21 required elements of a prescribed burn plan identified in the Interagency Prescribed Fire Planning and Implementation Procedures Reference Guide (U.S. Department of Agriculture and U.S. Department of the Interior, 2014).

The tutorial describes what is minimally acceptable for prescribed fire planning and implementation and uses the burn plan example in the RX-341 Prescribed Fire Plan Preparation course (Red Bull Prescribed Burn Plan).

The result of this tutorial is a burn plan document in Microsoft Word format that you can download to your local machine. For an example of what this burn plan looks like in Microsoft Word, see the Red Bull RX-341 Burn Plan topic, which you can view or download.

Important

Microsoft Word 2007 or later, or Word 2003 with the Microsoft Office Compatibility Pack, is required to open the burn plan document. You can download the compatibility pack at Microsoft.com.

Note that once you create a burn plan document in IFTDSS, it is stored within the system and can be re-used for subsequent documents.

There are also mapping tools within IFTDSS that can be used to develop vicinity and projectA project consists of one or more units that are organized and managed as a single endeavor. maps and can be used for hazard analysis and contingency planning. See Assessing Fire Hazard for Prescribed Burn Planning for information on using hazard analysis tools in support of your prescribed burn planning.

Exporting Mapping Data from IFTDSS To Google Earth

In this step, you export your map summary data to a file to be opened in Google Earth. After exporting your data, you will evaluate the results and identify potential fire hazards and risks.

   Note   

Google Earth is an application that you download and install on your local machine. You can download Google Earth here:

http://www.google.com/earth/download/ge/agree.html

To export your map summary data,

  1. Access the Exports tab (located in the upper left panel).

  2. Select Download Map Summary Data (KMLKeyhole Markup Language (KML), which is based on XML, is a file format used to display geographic data in an Earth browser (such as Google Earth, Google Maps, and Google Maps for mobile). For more information, see http://code.google.com/apis/kml/documentation/kml_tut.html.). A pop-up window appears.



       Note   

    Depending on which browser you are using, this pop-up window may look different. However the pop-up window looks, choose Open.

  3. In the pop-up window, choose Open with Google Earth.
  4. Next, again in the Exports tab, select Download Input Data (KML). The same pop-up window appears.
  5. Choose Open with Google Earth.
  6. The map will now be displayed in Google Earth.

Create a New Project

Organize your work by creating a project. In IFTDSS, a project is like a workspace. A single project can contain multiple modelA model is a set of mathematical relationships that describe an aspect of wildland fire (Andrews and Queen, 2001); generally, models are equations that are published in scientific literature. For information on creating a new project, see Creating a Project.

Manage My Projects

Once you have created one or more projects, you can access and manage those projects on the Active Projects page. There are two ways to access that page: the Manage My Projects link or the Projects tab.

Choosing either of these takes you to the Active Projects page. There, you can view and manage existing projects. You can also create a new project by accessing the Create New Project link on this page.

Manage My Data Sets

Use this menu option (or click directly on the Data tab) to access the page where you can upload and manage project data.

Search Published Projects

In future versions of IFTDSS, you will be able to publish and share projects with other IFTDSS users.

Find Other Users

Use this menu option (or access the Collaborate tab) to find other IFTDSS users.

Detailed Description of the Issue

On the Configure step for these pathways, if you select a data set that is smaller than your run area, the output values for each pixel in the area within the run set but outside your data set will be set as zeros (rather than no data).

The following images illustrate this issue. In the example used for these images, we chose a data set smaller than our run area. In the first example, the area within the run but outside the data set area is displayed with an overlay that seems to indicate there is zero probability of burning for the area represented by those pixels.  Instead, because there is no data, there should be no overlay at all.

In the second image, the spatial landscape data correctly shows data in the data set area and no data outside of the data set area.

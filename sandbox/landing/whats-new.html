Improved Help Center

Version 3.1.1 Released Oct 29 - Nov 5 2018

  • More wind options in LFB - select options to use gridded winds, upslope winds, downslope winds, or static winds.

  • Fuel moisture conditioning - When running models, condition fuel moistures with IFTDSS data, or upload your own weather stream data.

  • AOIs for Reports - Now generate reports constrained by an area of interest for any report (Auto97th, LCP summary, Model summary).

  • Copy run button - Easily copy, edit, and re-run models.

  • Quickly view model run inputs - In My Modeling Playground, click the 'view input' button under any model output file to view a dialogue box displaying all model inputs used to create that file.

  • Improved Help Center - More easily browse and find Help Center topics.

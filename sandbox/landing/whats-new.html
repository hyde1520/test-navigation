Version 3.1.0 Released 6/14/18

  • Fuels Treatment Monitoring Effectiveness (FTEM) feature now available.

  • Use the FTEM map-based interface to easily locate fires and treatments.

  • Monitor treatments and create reports for FTEM from the same interface as IFTDSS.

  • Updated Help Center topics to help you use the FTEM feature.

  • Improvements to the user interface for account creation and password reset.

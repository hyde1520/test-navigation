We Need Your Help!

IFTDSS is always improving! The app is being built using modern agile development principles. This means that we’re always improving the existing features and adding more functionality. To make this all work we need to hear from YOU! Please use the Ideas Exchange Forum to tell us what you'd like to see in future IFTDSS versions.

Training

We're committed to helping you learn how to get the most out of the IFTDSS web app. Please visit our webinars page to learn about webinars and sign up for one today!

Support

IFTDSS has a four pronged support system that includes in-app support text, a robust help center, user forums, and a ticket system to get the help you need when you're stuck.

Community

Building a community of knowledge is no small task. To help you get started we created the IFTDSS Forums. There is a Questions and Answers Forum and a ideas exchange for sharing suggestions.

How to Use FTEM

The FTEM interface is arranged to walk you through the FTEM process. Use the tabs to select a wildfire, select your treatment areas, enter monitoring data, review, and submit your data. Below, we walk through each stage of this process. The following steps assume you have an FTEM account. If not, visit the FTEM Setup Topic for instructions. Upon opening FTEM, you will see wildfires with a split view of Map Studio on the left, and a table view on the right.

FTEM interface controls

Step 1: Pick Your Wildfire

Pick a wildfire based on year and wildfire name:

  1. Enter a Wildfire Year, the year the wildfire was declared under control, or extinguished.
  2. (Optional) Filter your results by agency. If you are not an administrator or dual agency editor, this option may not be visible.
  3. (Optional) Use the wildfire search field to further constrain your results by entering the fire name.
  4. Left-click to select the desired fire name from the list.

    5. Need to de-select your fire? Move the mouse cursor to the Wildfire tab and click the black X to the left of the wildfire name.

  5. (Optional) Click the Zoom To button to zoom to the fire on the map, or the Table Summary button to see a summary of the interactions for that fire.

  6. Proceed to the Treatments Tab. You may also proceed to Treatments by clicking the Select Treatments button located to the left of the Zoom To button.

Attached files can be added or downloaded in later tabs (see the Attachments topic). Notice that each wildfire listed is followed by information, left to right, describing the number of acres, control date, monitor status, and the number of interactions.

Step 2: Select Treatments

In this step you will choose all the treatments for your agency that you know interacted or influenced the fire. Monitoring data for each of these will be added via the Monitoring tab. This step can be thought of as your "shopping cart" where you place the treatments needing monitoring, and remove the ones that don't.

  1. (Optional) If you know there are treatments that interacted with the wildfire, but do not see them here, click the Add Treatment(s) button to search and select treatments near the fire (This option allows you to choose a buffer distance to search within).

    2. By using the check box for one or more treatment, you may choose to enter data for treatments individually, or in batches. Note that when running batches some fields will still need to be revisited and filled individually for each treatment, such as time of burn and acres burned.

  2. If there are treatments that did not interact with the fire, use the check box to select them and click the Remove Treatment(s) button. If you need to add these back in later, you may do so by using the Add Treatments(s) button.
  3. Once you have all desired treatments, select them either individually by clicking the check box to the left of the treatment name(s), or by using the Select All button.

    Note the colored icons to the left of the treatment names. In this example the polygon icons indicate these treatments are polygons, and their red color indicates monitoring/data entry for them has not been started. Learn more about these symbols by visiting the FTEM Symbology Topic.

  4. Proceed to the Monitor tab. You may also get to Monitoring by clicking Monitor Treatment(s) button at the bottom right of the list.

Notice that each treatment listed is followed by information, left to right, describing the treatment ID, type of treatment, date of completion,agency, number of acres, control date, jurisdiction unit, and the number of interactions.

Step 3: Enter Monitoring Data
  1. Select the treatments for data entry by clicking the check box next to each treatment(s).
  2. Click the Enter Data button.
  3. A series of questions for the treatment(s) interaction appears. Notice that each blue heading expands to a list of specific questions. The first header, Interaction Details and Treatment Effects, is required. Enter data into each field. We briefly describe guidance for data in each field in the FTEM Required Questions Guidance Topic.
  4. Proceed to any optional fields and enter data. We briefly describe guidance for data in each field in the FTEM Optional Questions Guidance Topic.
  5. Once your data is entered, click Save and Close to proceed. If you want to save your partially completed work and resume the rest of the process later, click Save.

    6. Note that in this example two of three treatments are selected for data entry. You can choose to enter the same data for all your treatments, or like this example, enter data for some treatments, and later enter different data for other treatments. This can be useful if multiple treatments were affected, but burned under different conditions.

  6. While your treatment is still selected, you may also use the Attachments button to add any relevant maps, images, or supporting materials (for more detail, see the Attachments topic). Attachments can include:

    • MS Word documents (.doc, .docx) up to 5 Mb.
    • PDF documents (.pdf) up to 5 Mb.
    • Image files (.gif, .jpg, .png) up to 1.5 Mb.

  7. Proceed to the Complete tab.

Step 4: Complete the Process by Reviewing and Submitting your Data

In the Complete tab:

  1. Click the Table Summary button to generate a review table of treatments and data entered. You may also review attachments using the Attachments button. At this point, if you are satisfied with your treatment data, you may proceed to Step B. However, if you find a data entry that needs editing or additional information, you can use the tabs to navigate back, select treatments, and modify data.

  2. Click the Complete Wildfire Monitoring button to submit your data to the FTEM System.

At the bottom of the tab, the number of "completed", "in progress", and "not started" treatments displays.

Note the Complete Wildfire Monitoring is grayed out until all treatments are completed.

Exporting Raster Data from IFTDSS

Landscape data and Landscape Fire Behavior Model outputs can be downloaded as a Geo Tiff (.tif) raster dataset for use in other applications such as ArcMap. To download a raster dataset from IFTDSS:

  1. Navigate to My Workspace from the top navigation menu
  2. Select the file you would like to download

  3. Click Download

  4. A dialogue box will appear in which you may rename the file if desired
  5. When ready, click Download

  6. A confirmation box will appear verifying the raster has been downloaded to your local machine in .zip file format. The file will be saved in your browsers "Downloads" folder, or whichever folder you have specified for your browser to use.

  7. From your local machine, you may locate and copy the zip file, extract the contents, and store where needed.

  8. Once extracted, the file will be a GeoTif (.tif) with separate bands for each landscape feature or model output. (See the Topic on Opening IFTDSS rasters in ArcMap for more details if needed)

The GeoTiff landscape file download includes 12 bands. There are 4 in addition to the 8 that are visible within the IFTDSS application. The downloads also have different units than IFTDSS presents.

Band # Band Name Units - GeoTiff Units - IFTDSS (Reports)
1 Elevation meters feet
2 Slope degrees degrees
3 Aspect degrees degrees
4 Fuel Model categorical categorical
5 Canopy Cover % %
6 Stand Height meters * 10 meters
7 Canopy Base Height meters * 10 meters
8 Canopy Bulk Density kg/m3 * 100 kg/m3
9 EVT (Existing Vegetation Type) categorical N/A
10 EVC (Existing Vegetation Cover) categorical N/A
11 EVH (Existing Vegetation Height) categorical N/A
12 Zone (LANDFIRE Map Zone) categorical N/A
  • For instructions on how to reclass outputs such as flame length in ArcMap, see the Reclassing GeoTiff Features in ArcMap Topic.
  • For instructions on using the exported layers in FlamMap, see the help Topic on Loading GeoTiff Files into FlamMap 5.0.

My Workspace

My Workspace is where all your files in stored in IFTDSS, these include:

  • Landscape Files
  • Model Input
  • Model Output
  • Shapefiles
  • Reports
Layout of My Workspace

To the left of My Workspace, you will see:

  • Display: Lists the type of files currently contained in your Workspace.
  • Folders: Lists the folders contained within your Workspace.

The main feature of My Workspace is a list of your files. This list is referred to as the Data Table. Click on the row in the Data Table to see the actions you may take on that item.

Creating Folders

You may organize your workspace by creating or deleting folders for your project information.

  1. Create a folder by clicking the Create Folder option at the top of the My Workspace page.

  2. Give your folder a name and click Submit.

  3. Your new folder will now be visible in the Folders list.

Deleting Folders
  1. To delete a folder in My Workspace, hover over the folder name in the left panel. Note a red "x" will appear to the right of the folder name.

  2. Click the red "x".

  3. You will be asked to confirm your deletion, click "Yes".

Folders can only be deleted if it is empty. To delete a folder with contents, delete the contents first, then delete the folder.

Exporting Landscape and Fire Behavior Raster Data

Landscape data and Landscape Fire Behavior Model outputs can be downloaded as a Geo Tiff (.tif) raster dataset for use in other applications such as ArcMap. To download a raster dataset from IFTDSS:

  1. Navigate to My Workspace from the top navigation menu
  2. Select the file you would like to download

  3. Click Download

  4. A dialogue box will appear in which you may rename the file if desired
  5. When ready, click Download

  6. A confirmation box will appear verifying the raster has been downloaded to your local machine in .zip file format. The file will be saved in your browsers "Downloads" folder, or whichever folder you have specified for your browser to use.

  7. From your local machine, you may locate and copy the zip file, extract the contents, and store where needed.

  8. Once extracted, the file will be a GeoTif (.tif) with separate bands for each landscape feature or model output. (See the Topic on Opening IFTDSS rasters in ArcMap for more details if needed)

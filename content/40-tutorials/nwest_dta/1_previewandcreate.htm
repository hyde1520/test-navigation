Previewing and Creating a Landscape Using Map Studio

Previewing the Landscape

My first stop was in Map Studio to get a good look at the area I’m interested in, and create a landscape to start working from. To do this I: 

  1. Access Map Studio from the top navigation panel, visible on any page in IFTDSS.
  2. Use the Zoom buttons, and mouse, to find my area. The location search box at the top of Map Studio can also be helpful.
  3. Open the Basemap Gallery to get a more detailed baselayer.
  4. Select "Imagery with Labels".

Create the Landscape

Next, I want to create my landscape, which will serve as a starting point for the rest of my analysis.To do this I:

  1. Click on the Landscape Tools button at the bottom of the screen.
  2. Select the Create New tab to create a new landscape.
  3. Select Draw Mode, then drag across the area I’d like to analyze. Alternatively, you can follow along by inputting the coordinates below directly:
    • East: -115.5848
    • West: -115.8096
    • North: 43.8854
    • South: 43.7675
  4. Select the version of LANDFIRE data I want, in this case 2012.
  5. Select the fire behavior fuel model type (13 or 40), here I have chosen Fuel Model 40.
  6. Name my new landscape "Granite Creek LF2012 unedited." I included the name, Landfire version, and its editing/treatment status to make it easy to discern this landscape from others once I start editing.
  7. Choose or create a new folder for this landscape, this is where the landscape will be stored in My Workspace. Here I have created a "Granite Creek Treatments" folder
  8. Once everything looks good, I click the Create button and wait a minute for my landscape to process.

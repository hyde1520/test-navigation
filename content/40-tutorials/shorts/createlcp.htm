Creating a Landscape and Summarize

Landscapes in IFTDSS can be created in multiple ways. For example, you can create a landscape in Map Studio using the Landscape Tools button . You can also create a landscape from most of the IFTDSS tasks, using the Create Here button as displayed in the example below.

For this example, weâ€™ll create a new landscape using that Create Here button from the Landscape Summary Task in IFTDSS. This serves two purposes, to create a landscape, and to do so in the interface that allows you to easily create a landscape summary if you so choose (covered in the next tutorial).

To get to the Landscape Summary interface:

  1. Navigate to the Planning Cycle using the top navigation menu.
  2. Select the Landscape Evaluation stage of the Planning Cycle.
  3. Click the Landscape Summary task.

From the Landscape Summary page, click Create Here:

Next, you'll go through the steps of creating a landscape. As you go through the process below, use the following values:

  • Location: Tahoe National Forest, location bounds in degrees: West: -120.8856 , East: -120.6044 , South: 39.4463 , North: 39.6149
  • Landfire version: 2012
  • Fuel Model: Fuel Model 40
  • Landscape Name: HaypressCrkLf2012unedited
  • Folder: Specify or create: Haypress Creek Project

To help keep track of your data, we recommend including the LANDFIRE version in the name.

  1. Use the bounding box fields and enter the coordinates, in degrees, for the west, east, south, and north bounds of your area.

    2. The maximum area for a landscape is 12,188,160 acres (138 miles x 138 miles).

  2. Choose the version of LANDFIRE data you would like to use from the dropdown list
  3. Choose the fire behavior fuel model (Fuel Model 40 or Fuel Model 13) to include in your landscape file from the dropdown list.
  4. Give your landscape a descriptive name, this will be helpful as you accumulate more landscape files in IFTDSS.
  5. Select the folder where you would like to store your landscape file, or click + Folder to create and name a new folder to place your landscape file.
  6. Click Create to generate your landscape file.

Once you click Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, displays the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass is replaced by a green checkmark. Use the Refresh button next to the landscape name to see the green checkmark.

For the rest of your activities in IFTDSS this landscape will be available in the folder you specified in My Workspace, and access it in dropdown menus throughout IFTDSS.

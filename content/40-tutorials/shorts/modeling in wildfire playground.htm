Modeling Landscape Fire Behavior in Modeling Playground

It can often be helpful to run several different scenarios before going through the process of developing treatment alternatives. Modeling Playground is a space to run these scenarios without having to do so as part of a specific task in IFTDSS.

To familiarize yourself with this part of IFTDSS, run a fire behavior scenario for the HaypressLF2012unedited landscape:

 

  1. Click the Playground link in the top navigation
  2. Select 'Landscape Fire Behavior' from the dropdown menu
  3. Click Run. Note that the model runs from the landscape summary are also available in this interface, just like in MyWorkspace.

    4. All saved runs are stored in the same folder as your selected landscape, or in the ‘Playground folder’ if a new landscape is created but no folder specified.

Populate the input fields and run the model:

  1. Give your run a descriptive name. In this case 'Haypress_cool_wet'
  2. Select the HaypressCrklf2012unedited landscape from the dropdown menu. Notice that your landscape details are displayed below this area once you select your landscape.

    3. Next, select the input type, starting with 'Wind', and fill in the required fields and run the model:

    For this example you may use any values you choose.

  3. Specify windspeed, in miles per hour, at 20 feet above the ground or above the vegetation (allowed values 0-100 mph).
  4. Enter wind direction as the direction the wind is blowing from (in degrees clockwise from north, with north being 0 or 360 degrees). Alternatively you may enter -1 for upslope wind, or -2 for downslope wind.
  5. Specify a crown fire calculation method
  6. Enter a foliar moisture value
  7. Specify fuel moisture, in percent, for:
    • 1 hour fuels (1hr FM) - allowed values 2-100
    • 10 hour fuels (10hr FM) - allowed values 2-100
    • 100 hour fuels (100hr FM) - allowed values 2-100
    • Herbaceous fuels (Herb) - allowed values 30-250
    • Live Woody fuels (WoodyFM) - allowed values 60-200

    If you would like to specify fuel moisture by fuel model, you could do so by adding rows, click + add row. This will would a new line with a fuel model dropdown list where you could specify your fuel model and enter moisture data for that fuel model. You may repeat this as many times as needed.

  8. Upon completing the inputs, click the Save and Run button. You will notice a Save button is also available for the times when you need to close a run before completing the inputs, but want return to it later.

Upon the running the model, you will be taken back to the Modeling Playground page

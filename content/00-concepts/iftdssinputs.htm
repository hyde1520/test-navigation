Inputs in IFTDSS

You may start using IFTDSS with no inputs simply by downloading the data needed and populating model parameters. However, in most cases, you will have some data on your landscape already. Here is a list of the inputs used within IFTDSS:

Landscape Files (.lcp)

Landscape files are the basic file for using IFTDSS, and drive fire behavior and many fire effects modeling. You may create a landscape file from within IFTDSS from LANDFIRE. .

Shape Files (.shp)

In some instances, such as defining a treatment area, you may want to identify the area using a shape file. If you already have a shapefile it may be uploaded to IFTDSS.

Fire Behavior Fuel Models

Fire behavior fuel model data are contained within the Landscape file (see above)

Model parameters

Modeling parameters such as moisture, windspeed, etc. are populated when you choose a specific model to run within IFTDSS. These appear as blank fields within the application. If you enter a value that is outside the range used by a specific model, the value will be highlighted and you will be instructed to edit that value before proceeding.

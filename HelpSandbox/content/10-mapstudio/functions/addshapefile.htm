Add a Shapefile

Use the Add Shapefile to upload a shapefile to the IFTDSS Map Studio.

Upload a shapefile

Click Upload File.

Choose your zipped shapefile from your local drive.

The .zip file must contain the .shp, .shx, .dbf, and .prj files that comprise the shapefile.

Select the desired fields from your shapefile using the checkbox next to each field name (Up to 10).

Click Upload File.

Click Add to Map.

Next you will save your shapefile.

Saving Shapefiles for Future Use

Follow the steps below to permanently save an uploaded shapefile in My Workspace for later use in IFTDSS. Without this step, navigating away from Map Studio results in an uploaded shapefile not being saved. This is a known issue that will be corrected in a later IFTDSS release.

To retain your uploaded shapefiles in IFTDSS if you follow the steps below:

A) Upload a zipped shapefile, ensure you select which attributes you would like to include (you may choose up to 10). A zipped shapefile must contain .shp, .shx, .dbf, and .prj files.

B) When completed, a confirmation message will appear to verify the shapefile has been uploaded.

C) To select the new shapefile, exit Add Shapefile and open Layer List to zoom to your new shapefile:

  1. Click the icon to the right of your shapefile and
  2. Click the Zoom to option

D) Proceed to the next dropdown step, either 'Save a Single Uploaded Shapefile' or 'Save Multiple Shapefiles'

Save a single uploaded shapefile:

  1. Click the Copy from Existing Features button ,
  2. Make sure the Select Single Part option is selected
  3. Select the polygon you wish to save (selected polygons will turn yellow). This can be done by either left clicking the individual polygon.
  4. Give your polygon a descriptive name.
  5. Choose a folder in which to save your polygon.
  6. Click Create.
Save multiple polygons:

  1. Click the Copy from Existing Features button ,
  2. Make sure the Select multipart option is selected, then
  3. Left mouse click your first polygon, you will notice its color turns to yellow.
  4. Next click Copy from Existing Features button again and left click on polygon two, repeat these steps for each polygon until they are all outlined in yellow.
  5. Next, type a name for your polygons in the edit box.
  6. Specify the folder to save the set in.
  7. Click Create.

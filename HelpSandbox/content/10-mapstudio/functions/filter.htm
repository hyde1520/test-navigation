Filter Button

The Map Studio Filter button enables you to limit the visibility of polygons or shapefiles when working in Map Studio. This can be useful if you are working in an area with many shapefiles (such as many treatments clustered in a small area, or overlapping each other). Only the polygons that meet the expression criteria will be visible in the map.

To add a filter, first go to the Edit Filter tab. You may filter by folder, file name, or both. When a specific folder is chosen using the pick-list, only those within the folder are listed.

To further select the shapefiles that are viewable as you create your filter, click the Select button to the right of the shapefile list, and select or unselect specific files. If you would like to select all the shapefiles click the Select All button, or to deselect all selected shapefiles click the Deselect button. After you have selected the shapefiles you would like to display click Apply Filter .

This filter can be removed anytime by going back to the Filter button and selecting Remove Filter .
